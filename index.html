Política de Privacidad

Escudo Urbano

Última actualización: Marzo 2026

1

Información General

Escudo Urbano es una aplicación de seguridad comunitaria desarrollada para facilitar la comunicación y respuesta ante emergencias en el entorno urbano.

Esta política describe cómo recolectamos, usamos y protegemos tu información personal. Al usar la aplicación, aceptas las prácticas descritas en este documento.

2

Datos que Recolectamos

Datos de registro

  • Número de teléfono (verificado mediante OTP)
  • Nombre de usuario

Datos de ubicación

  • Ubicación en tiempo real para alertas de emergencia y visualización en mapa comunitario
  • Se accede a la ubicación en segundo plano únicamente durante eventos de emergencia activos

Datos del dispositivo

  • Identificador del dispositivo
  • Sistema operativo y versión
  • Token de notificaciones push (Firebase Cloud Messaging)

Datos de uso

  • Registro de alertas enviadas y recibidas
  • Historial de incidentes reportados
3

Cómo Usamos tus Datos

Dato Uso
Número de teléfono Autenticación OTP y notificaciones WhatsApp vía Twilio
Ubicación Mostrar alertas cercanas y enviar tu posición en emergencias
Token FCM Envío de notificaciones push de alertas comunitarias
Datos de uso Mejorar el servicio y análisis de incidentes
4

Permisos del Dispositivo

  • Ubicación en primer y segundo plano: para activar alertas de emergencia con tu posición
  • Sensores (acelerómetro): para la función shake-to-activate de emergencia
  • Internet: para sincronización en tiempo real con el servidor
  • Notificaciones: para recibir alertas comunitarias en tiempo real
5

Servicios de Terceros

Escudo Urbano utiliza los siguientes servicios de terceros. No vendemos ni compartimos tus datos con fines comerciales.

Servicio Propósito Política
Firebase (Google) Autenticación, base de datos, notificaciones push Ver política
Twilio Notificaciones WhatsApp de emergencia Ver política
6

Retención de Datos

  • Los datos de cuenta se conservan mientras la cuenta esté activa
  • Los registros de alertas se conservan por 90 días
  • Al eliminar tu cuenta, todos los datos son eliminados en un plazo máximo de 30 días
7

Seguridad

  • Comunicaciones cifradas mediante HTTPS/TLS
  • Autenticación de dos pasos mediante OTP
  • Acceso restringido a la base de datos Firebase mediante reglas de seguridad
  • Sin almacenamiento local de contraseñas ni datos sensibles
8

Derechos del Usuario

Puedes en cualquier momento:

  • Acceder a tus datos desde la configuración de la app
  • Eliminar tu cuenta y todos los datos asociados
  • Revocar permisos de ubicación desde la configuración de tu dispositivo
  • Solicitar una copia de tus datos contactándonos directamente
9

Menores de Edad

Escudo Urbano no está dirigida a menores de 13 años. No recolectamos intencionalmente datos de menores de edad. Si detectamos que un menor ha proporcionado información personal, la eliminaremos de inmediato.
